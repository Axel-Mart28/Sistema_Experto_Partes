4. Tratamiento del Conocimiento (Motor de Inferencia)
¿Qué es?
Es el "cerebro lógico" o el núcleo de procesamiento del sistema experto.

¿Para qué sirve?
Para deducir nuevas piezas de información, diagnosticar problemas o tomar decisiones automáticas. Combina las reglas fijas con los datos actuales para generar conclusiones operativas.

¿Cómo funciona?
Toma los datos alojados en la Base de Hechos y los contrasta sistemáticamente contra las reglas alojadas en la Base de Conocimiento. Utiliza métodos de inferencia (determinísticos o probabilísticos) para ejecutar acciones o llegar a un veredicto. Si no hay datos suficientes, puede solicitar más información.

Ejemplo Práctico:
El motor lee el dato actual ("variación de 0.6 grados") y lo evalúa contra la regla de tolerancia ("límite de 0.5 grados"). Infiere de manera automática que la prueba falló debido a una anomalía mecánica y ejecuta la orden de detener el ciclo de prueba.
